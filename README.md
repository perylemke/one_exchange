**One Exchange**
===================
One Exchange is an application that convert currency.

It's a first app to a [One Bit Code Bootcamp](http://onebitcode.com/) and a so cool!

### Technologies used
- Ruby 2.3.3
- Rails 5.0.1
- PostgreSQL
- Docker
- Docker Compose
- Codeship
- Heroku
